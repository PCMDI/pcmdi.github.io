Request for AMIP II Boundary Conditions Data on a specified grid 


Please email the request information to:
  1. Karl Taylor - taylor13@llnl.gov
  2. Peter Gleckler - gleckler1@llnl.gov
  3. Charles Doutriaux - doutriaux1@llnl.gov

Indicate in the request the following:
    1. Grid type (Uniform or Gaussian)
    2. Spherical Harmonic Truncation (e.g., T42, R31, etc.)
    3. Resolution:
      1. # of longitudes
      2. # of latitudes
    4. First grid point:
      1. longitude
      2. latitude
    5. File format (NetCDF COARDS convention, ASCII, GRIB, DRS)
    6. Name of your institute
    7. Model/Version name
    8. Your name
    9. Your email address
    10. Questions or comments:
      1. For a grid that is neither Uniform nor Gaussian, provide a description.
      2. If applicable, specify how we can access a NetCDF version of your grid (ftp site).
