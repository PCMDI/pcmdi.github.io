Laboratoire de Météorologie Dynamique: Model LMD LMD6s (3.6x5.6 L11) 1995

Model LMD LMD6s (3.6x5.6 L11) 1995 is identical to its companion model LMD LMD6b (3.6x5.6 L11) 1995 except that the SECHIBA land-surface model replaces the "bucket" scheme for treatment of soil hydrology and land-surface evaporation. Model LMD LMD6s (3.6x5.6 L11) 1995 differs from the AMIP baseline model LMD LMD5 (3.6x5.6 L11) 1991 in the following respects:

  • Model Designation
  • Model Lineage
  • Model Documentation
  • Numerical/Computational Properties
    • Horizontal Representation
    • Computer/Operating System
    • Computational Performance
    • Initialization
    • Time Integration Scheme(s)
    • Sampling Frequency
  • Dynamical/Physical Properties
    • Atmospheric Dynamics
    • Solar Constant/Cycles
    • Radiation
    • Cloud Formation
    • Precipitation
    • Surface Characteristics
    • Surface Fluxes
    • Land Surface Processes

  • References
