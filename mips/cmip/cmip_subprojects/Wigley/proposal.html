    The aim of this project is to examine and understand the lagged correlation structure of NH and SH temperatures (r(N(i), N(i-n)), r(S(i), S(i-n)) and r(Ni(i), S(i-n))) for different lags (n = -30 to +30, specifically). We plan to compare the correlation structure observed temperatures with that for O/AGCM control-run data, and to inter-compare the structures for different control runs. Preliminary work with the GFDL control run indicates that the difference between observed and control-run data can be explained by external forcing effects. When these effects are subtracted from the observed data, the results agree extremely well with the control-run. This provides an interesting validation of the control run, and, at the same time provides strong support for a significant anthropogenic forcing effect on near-surface temperatures.
