Report 24:    Documentation of the AMIP Models on the World Wide Web

    Phillips, Thomas J.
    August 1995, 14 pp. 
    Summary documentation of the numerics, dynamics, and physics of models participating in the Atmospheric Model Intercomparison project (AMIP) is now available on the Internet's World Wide Web. This report describes the principal attributes of the electronic model documentation and provides instructions on how to access it.Hypertext, (pdf).
